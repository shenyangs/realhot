# 品牌客户讲解版流程图设计说明

## 视觉方向

这张图不是给技术团队看系统结构，而是给销售或客户成功同事对品牌客户讲“我们怎么把内容做出来”。所以重点不是接口、模型、数据库，而是把客户能听懂的生产闭环讲清楚：

1. 先吃进品牌资料和市场信号
2. 再判断这件事值不值得做
3. 通过人工确认，确保方向可控
4. 用 AI 快速产出图文、视频、口播、字幕首版
5. 在工作台完成人工精修
6. 最后导出、发布，并把结果回流

## 设计语言

整体采用“米色底 + 深绿主轴 + 橙色人工节点 + 深色交付节点”的方式。这样能让客户一眼看出：

- 绿色是系统自动推进
- 橙色是人工判断和把关
- 深色是最终业务交付

画面结构采用“两路输入汇总到一条主生产线”的方式，强调这套工作台不是凭空写内容，而是先吃品牌上下文、再吃市场信号，再进入判断与制作。

## 销售讲解口径

推荐讲法：

“我们不是让 AI 直接乱写，而是先把品牌资料和外部热点喂进去，再由系统先做机会判断。方向确认后，一键生成图文、视频、口播和字幕首版，最后仍然保留人工精修和审核，所以速度更快，但品牌风险依然可控。”
